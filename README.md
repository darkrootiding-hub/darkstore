APK Store – Premium Animated UI
A modern, mobile-first APK Store UI built using HTML, CSS, and JavaScript, featuring smooth animations, glassmorphism design, and Play Store–inspired UX patterns.
This project focuses on frontend UI/UX, performance, and interaction design.
Overview
This project demonstrates how a professional app-store–style interface can be built without frameworks.
It includes advanced UI components such as animated cards, modal bottom sheets, filtering systems, and responsive layouts optimized for mobile devices.
⚠️ This is a UI + frontend logic project, not a real APK distribution platform.
